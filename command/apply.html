# terraform apply

apply 當確認都已經 final 後，apply 會按照設定執行更新，而 apply 更新有兩種模式。

$ terraform apply
  • 按照 *.tf 的設定進行更新 (預設)
  • 使用 plan -out 產生的異動檔更新。

plan -out 不會詢問 (yes/no) 而是直接執行。
